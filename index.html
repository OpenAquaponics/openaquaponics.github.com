Introduction

For those who are not familiar with setting up hardware and software systems, don't worry. I'll try my best to keep everything as stright forward and simple as possible.

The following are greate reference instructions in case I missed a step or if something doesnâ€™t work as listed: http://www.raspberrypi.org/downloads http://elinux.org/RPi_Easy_SD_Card_Setup http://www.penguintutor.com/linux/raspberrypi-webserver

This how-to is going to show you how to setup your Raspberry Pi to host both the OAServer and OANode applications. This first iteration is going to assume you are running both applications on the same device. There is nothing stopping you from hosting OAServer on another Linux machine, but for the purposes of this how-to, I will assume you want to run everything on one device. Eventually there will be a "cloud" instance of OAServer, and you won't need to worry about hosting that software unless you would like to run a local system. So onward!!

What you need

To get started you will need:

  • Computer with SD card reader
  • Raspberry Pi
  • SD card (4GB or more)
  • Ethernet cable (for the initial setup)
  • Monitor (requires HDMI)
  • Keyboard

Download and Install Raspbian

So we begin the tutorial by downloading and installing the Raspberry Pi image. The following website has great step-by-step instructions on how to do this. http://elinux.org/RPi_Easy_SD_Card_Setup#Copying_the_image_to_an_SD_card_on_Windows

Unplug the SD card from your machine and place it in the Raspberry Pi. Go ahead and plug in a monitor, keyboard, and power supply.

On first boot execute the following:

sudo raspi-config

, enable ssh

run sudo raspi-config

execute update optional execute enable boot_behaviour optional execute expand_rootfs (depending on SD card size, this can take a long time)

Finish and reboot

use your router to get IP address of device putty ip of your device

Optional If you really want to latest run: sudo apt-get upgrade

Install Developer Packages

Install a command line text editor of your choice. There are a multitude of options, choose whichever you prefer. If you are not sure, nano is an easy one to use.

sudo apt-get install vim
sudo apt-get install nano
sudo apt-get install emacs

Go ahead and make sure these applications are installed. NTP syncs the system clock and CURL allows you send HTTP messages from the command line.

sudo apt-get install curl ntp

Now of course you are going to need the GIT client, so go ahead and install it.

sudo apt-get install git

Install Optional Packages

The following packages are optional and are handy to have installed on the system. 

sudo apt-get install locate
sudo apt-get install ack
# Move these to OAServer

Install LAMP Packages

This will setup the RPi to have a webserver and database running. OAServer requires this in order to run Install the Apache server

sudo apt-get install mysql-server
sudo apt-get install apache2
sudo apt-get install php5 php5-mysql php5-curl
sudo apt-get install phpmyadmin
  < Follow the on screen instructions for passwords, you will need MYSQL_PASSWD >

NOTE: You may run into problems during the installation of mysql if you SD card is too small. I'd recommend at least a 8GB SD card. http://databaseblog.myname.nl/2013/01/how-to-install-mysql-succesfully-on.html NOTE: Go get a drink. This can take a while.
